# 🚀 SmartBiz AI

> An AI-powered Sales & Business Assistant for Small Businesses built using Flask, SQLite, Google Gemini AI, HTML, CSS and JavaScript.

---

## 📌 Overview

SmartBiz AI is an intelligent sales management system designed for small shop owners and local businesses.

Instead of only recording sales, it helps business owners understand their business using Artificial Intelligence.

Users can:

- Record daily sales
- View analytics
- Track profit
- Generate reports
- Ask questions in natural language using Gemini AI

The project combines traditional business management with Generative AI to make business insights available to everyone.

---

# ✨ Features

### 📊 Interactive Dashboard

- Today's Sales
- Orders Count
- Estimated Profit
- Best Selling Product
- Recent Sales
- Sales Charts

---

### ➕ Sales Management

- Record new sales
- Automatic total calculation
- Delete sales
- Store all records in SQLite Database

---

### 🤖 AI Business Assistant

Powered by **Google Gemini AI**

Ask questions like:

- What sold best today?
- Which products generated the highest revenue?
- Compare today's sales
- Suggest ways to increase profit
- Which products should I restock?

The assistant understands natural language and provides intelligent business insights.

---

### 📈 Reports

Generate graphical reports including

- Sales by Product
- Revenue Distribution
- Performance Analysis

---

### 🌐 Bilingual Ready

Supports

- English
- Tamil

making it suitable for local shop owners.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/aradhanaaru/smartbiz-ai.git
```

Move into project

```bash
cd smartbiz-ai
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run application

```bash
python app.py
```

Visit

```
http://127.0.0.1:5000
```
